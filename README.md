


## E-commerce - TT Tipis

## Description

This is a simple e-commerce project that handle adding products, create orders and track them !
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# seed the product table
$ npm run seed
```


### Navigate To : 
```
  http://localhost:3000/api
```
